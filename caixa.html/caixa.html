ANATOMIA DE UMA BOX

  • Width = largura
  • Height = altura
  • Border = borda
  • Borda para dentro é padding = espaço da borda
  • Borda para fora é margin = margem
  • Outline = contorno

SIGNIFICADOS

box-level

Ele pula para a linha de baixo e vai desenhar um box level, ele oculpa a largura total da tela.

inline-level

Ele pula para o lado e desenha a caixa exatamente o tamanho que ele precisa e da continua na mesma linha.
